Alleged Game Features

  • 🎲 Random pipe placement that is possibly unfair.
  • 🐦 Tap to control a bird (sometimes it works!).
  • 📈 Scoring system: How far can you struggle before it all goes wrong?
  • 💀 Game-over screen with your "final" score (no refunds).

What Even Is This?

Ugráló Madár is like Flappy Bird but without all the polish. This was made using Python and Pygame in what can only be described as a fever dream.
